# Platform Policies

This folder stores Layer 5 platform policies grouped by policy type.

## Rules

- Group folders represent the policy type code `XX`.
- Policy files follow `PP-XX.YY-{slug}.md`.
- Each group must include `runtime.time.md` to map effective platform policies to Layer 3 runtime policies.
- Mapping direction is inward only: Layer 5 -> Layer 3.
